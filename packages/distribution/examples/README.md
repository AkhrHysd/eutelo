# Examples（参考資料）

このディレクトリには、Eutelo ドキュメント構造を活用する際の
**参考例（例示のみ）** が含まれています。

> **Note:**  
> 本パッケージでは実装コードは含まれていません。  
> 実際の CI/CD や scaffold ツールの実装は別途必要です。

---

## ディレクトリ構成

```
examples/
  ci/              # CI/CD ワークフローの例示
  scaffold/         # プロジェクト構成の例示
  README.md         # このファイル
```

---

## 利用方法

各ディレクトリの README またはサンプルファイルを参照してください。

- **CI 例**: `ci/github-actions-workflow-example.yml` を参照
- **Scaffold 例**: `scaffold/README.md` を参照

---

## 注意事項

- これらの例は **参考用** であり、そのまま使用できる実装ではありません
- 実際のプロジェクトでは、要件に応じて調整・実装が必要です
- 実装に関する詳細は、各プロジェクトのドキュメントを参照してください
